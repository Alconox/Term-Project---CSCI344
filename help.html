User Guide

Use the bars at the top of any page to navigate to other areas. Everything is self-explanatory for the most part. If you have other comments or questions about this application, please email us. Do not wait until the last minute to complete projects!


Technologies Used

  • HTML: HTML stands for Hypertext Markup Language. It is a computer language that is used for website creation. It can be used to link several documents in order to display for the user.
  • CSS: CSS stands for Cascading Style Sheets. It can be used to describe html. With this you can have more detail in your page by defining style, layout, and more.
  • ANGULAR: Angular is a framework to help with the use of client side applications. It can work with HTML for visual advancement as well as be the bridge between the client side and the backing of a website.
  • MONGODB: MONGODB is a database program. It can be used for keeping track of information that can be used throughout a system. It can be used as a file system.
  • NODE.JS: NODE.JS is a javascript platform that can be used for network applications. Runtime development on the server side.
  • Express: Express is a application framework within node that can be used and built upon. Can be used for web applications. A common use of Express is that of a server. This can be used as the backing of a website.
  • JQuery: JQuery is a JavaScript library. It makes API much simpler to use

Source Files

  • index.html - The index page is the first page the user sees and their gateway to the rest of the site.
  • index.js - The index.js file loads a server using Express that loads the index.html to the localhost. It also creates a database using Mongoose that inputs data using node.js patterns.html
    • /categories/angular.js - Connects Mongo database to pattern pages, to supply data.
    • /categories/*.html - All HTML files within the categories folder exist to display patterns within their specific categories: hats, sweaters, etc.
    • /categories/data.json - Backup JSON file for data, to draw data from if server is inaccessible.
    • /css/style.css - The file that controls the styling for the site, including for pattern display.
    • /scripts/getPattern.js - A JavaScript function that retrieves data using JQuery and AJAX and inputs it into divs in each pattern page, filtered by category.
    • /scripts/maps.js - JavaScript code for displaying Google Maps.
    • /scripts/navigation.js - JavaScript functions that open and close menus across the site.
    • /scripts/slideshow.js - JavaScript function that controls the carousel on the main page.
